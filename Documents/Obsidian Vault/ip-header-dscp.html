✦ 公式 ✦
DSCP = Class × 8 + Drop × 2
为什么是 ×8？因为 Class 在 bit 5,4,3 位置，2³ = 8
为什么是 ×2？因为 Drop 在 bit 2,1 位置，2¹ = 2

Class ↑ = 转发优先级更高（Class 4 > Class 3 > Class 2 > Class 1）
